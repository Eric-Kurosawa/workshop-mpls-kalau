# Workshop MPLS
**Lab feito no EVE-NG do Workshop de MPLS ministrado pelo Gustavo Kalau**

<br>

**Passos realizados:**
1. Exclusão do processo OSPF de cada Roteador
2. Configuração do OSPF e MPLS no núcleo da rede (AS65007)
3. Configuração do VRF nos Roteadores PE1, PE2 e PE3
4. Configuração do iBGP no CORE entre os PEs
5. Configuração do eBGP entre os CEs e os PEs (e RD nos PEs)
6. Configuração do RT e L3VPN nos PEs

<br>

**Legenda:**
- **AS:** Autonomous System
- **VRF:** Virtual Routing and Forwarding
- **PE:** Provider Edge
- **CE:** Customer Edge
- **RD:** Route Distinguisher
- **RT:** Route Target
